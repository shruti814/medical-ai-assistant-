# medical-ai-assistant-
Built an ai which makes the taks of doctors easier, by recording their conversations, extracting symptoms, and predicting possible diagnosis
Main aim of creating this project was to help clinics which have a large number of patients, and sometimes the doctor gets exhausted too
Because of the mental fatigue, wrong diagnosis can also be prescribed, and to support the medical decision this project was made.
This AI works as a supporting agent, the final verdict is given of the doctor
